На данной вкладке вы можете указать настройки для сайта.

Чтобы открыть данную вкладку, выберите **Клиенты** → **Сайт** в боковом меню и выберите вкладку **НАСТРОЙКИ**.

![image1](/img/ru/admin_site_settings/image1.png "Настройки") <center><u>Настройки - фото 1</u></center>

## Доступные настройки

Доступны следующие настройки страницы:

![image2](/img/ru/admin_site_settings/image2.png "Настройки") <center><u>Настройки - фото 2</u></center>

|  №  | Настройки |
| :-: | --------- |
| 1 | Выберите из списка **Сайт**, настройки которого вы хотите изменить  |
| 2 | Укажите **Цвет ссылок** |
| 3 | Укажите **Цвет блока навигации** |
| 4 | Укажите **Высоту логотипа** (в пикселях) |
| 5 | Укажите ссылку на **Логотип** |
| 6 |  |
| 7 |  |

Нажмите **ВЫПОЛНИТЬ**, чтобы сохранить настройки.

![image3](/img/ru/admin_site_settings/image3.png "Настройки") <center><u>Настройки - фото 3</u></center>
