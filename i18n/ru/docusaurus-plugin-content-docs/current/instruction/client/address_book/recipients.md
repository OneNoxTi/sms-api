На данной вкладке вы можете создавать и удалять адресные книги, содержащие номера телефонов получателей. Вы также можете добавить в адресную книгу дополнительную информацию, например, имя или день рождения.

Чтобы открыть данную вкладку, выберите **SMS** → **Адресная книга** в боковом меню и перейдите на вкладку **ПОЛУЧАТЕЛИ**.

![image1](/img/ru/client_address_book_recipients/image1.png "Получатели") <center><u>Получатели - фото 1</u></center>

## Добавление адресной книги

Чтобы добавить новую адресную книгу, нажмите кнопку **+ ДОБАВИТЬ**.

![image2](/img/ru/client_address_book_recipients/image2.png "Получатели") <center><u>Получатели - фото 2</u></center>

Далее выполните следующие действия:

|  №  | Действие |
| :-: | -------- |
| 1 | Введите **Название** адресной книги |
| 2 | Выберите **Основную** адресную книгу из имеющихся. Это поможет вам сгруппировать их |

![image3](/img/ru/client_address_book_recipients/image3.png "Получатели") <center><u>Получатели - фото 3</u></center>

Далее добавьте **Получателей** [вручную](#добавление-вручную) или [загрузите](#загрузка-из-файла) получателей из **файла**.

После добавления номеров получателей нажмите кнопку **ВЫПОЛНИТЬ**.

![image4](/img/ru/client_address_book_recipients/image4.png "Получатели") <center><u>Получатели - фото 4</u></center>

### Добавление вручную

Чтобы добавить получателей, введите их телефонные номера в поле **Получатели**.

![image5](/img/ru/client_address_book_recipients/image5.png "Получатели") <center><u>Получатели - фото 5</u></center>

Номер каждого получателя указывается с **Новой строки**, например:

![image6](/img/ru/client_address_book_recipients/image6.png "Получатели") <center><u>Получатели - фото 6</u></center>

Чтобы добавить дополнительную информацию о получателе, введите ее через точку с запятой [ ; ], например:

![image7](/img/ru/client_address_book_recipients/image7.png "Получатели") <center><u>Получатели - фото 7</u></center>

### Загрузка из файла

Для загрузки списка получателей из файла сначала выберите тип файла в раскрывающемся списке.

![image8](/img/ru/client_address_book_recipients/image8.png "Получатели") <center><u>Получатели - фото 8</u></center>

|  №  | Тип | Описание |
| :-: | --- | -------- |
|  1  | **CSV (разделитель - точка с запятой)** | CSV файл с данными, разделенными точкой с запятой |
|  2  | **CSV (разделитель - запятая)** | CSV файл, в котором данные разделены запятой |
|  3  | **TXT (разделитель - табуляция)** | TXT файл, в котором данные разделены табуляцией |
|  4  | **XLS/XLSX (Excel документ)** | Формат XLS или XLSX, стандартный файл Excel с данными в отдельных ячейках |

После выбора типа файла нажмите **ВЫБЕРИТЕ ФАЙЛ** и выберите файл на вашем устройстве.

![image9](/img/ru/client_address_book_recipients/image9.png "Получатели") <center><u>Получатели - фото 9</u></center>

## Поиск

Вы также можете найти получателя по **Номеру телефона** или **Названию**. Чтобы найти получателя, выполните следующие действия:

|  №  | Действие |
| :-: | -------- |
| 1 | Укажите параметр поиска: <li>**По номеру телефона**</li> <li>**По названию**</li> |
| 2 | Укажите **Номер телефона** или **Название** для поиска получателя |

![image10](/img/ru/client_address_book_recipients/image10.png "Получатели") <center><u>Получатели - фото 10</u></center>

## Список адресных книг

Все добавленные адресные книги перечислены в алфавитном порядке.

![image11](/img/ru/client_address_book_recipients/image11.png "Получатели") <center><u>Получатели - фото 11</u></center>

Справа от названия адресной книги указывается количество номеров получателей содержащихся в ней.

![image12](/img/ru/client_address_book_recipients/image12.png "Получатели") <center><u>Получатели - фото 12</u></center>

## Удаление адресной книги

Чтобы удалить адресную книгу, нажмите иконку кнопки **Удалить** справа от нужной строки адресной книги.

![image13](/img/ru/client_address_book_recipients/image13.png "Получатели") <center><u>Получатели - фото 13</u></center>

Откроется всплывающее окно, в котором нужно подтвердить удаление.

![image14](/img/ru/client_address_book_recipients/image14.png "Получатели") <center><u>Получатели - фото 14</u></center>

## Просмотр адресной книги

Чтобы просмотреть адресную книгу нажмите на нее в списке адресных книг.

![image15](/img/ru/client_address_book_recipients/image15.png "Получатели") <center><u>Получатели - фото 15</u></center>

Открывается страница выбранной адресной книги, которая визуально разделена на два блока:

![image16](/img/ru/client_address_book_recipients/image16.png "Получатели") <center><u>Получатели - фото 16</u></center>

|  №  | Блок | Описание |
| :-: | ---- | -------- |
| 1 | **Информация про адресную книгу** | Содержит информацию про адресную книгу |
| 2 | **Номера получателей** | Содержит список номеров получателей, позволяет добавлять, удалять и экспортировать номера получателей |

### Информация об адресной книге

![image17](/img/ru/client_address_book_recipients/image17.png "Получатели") <center><u>Получатели - фото 17</u></center>

|  №  | Данные | Описание |
| :-: | ------ | -------- |
| 1 | **ID адресной книги** | Уникальный идентификатор адресной книги |
| 2 | **Название адресной книги** | Название адресной книги <br/> Для редактирования введите новое имя в это поле и нажмите иконку кнопки **СОХРАНИТЬ** справа от поля ввода |
| 3 | **Основной список** | Название родительской адресной книги |
| 4 | **Количество номеров** | Количество номеров в данной адресной книге |

### Номера получателей

#### Добавление номера

Чтобы добавить номер в адресную книгу, нажмите **+ ДОБАВИТЬ**.

![image18](/img/ru/client_address_book_recipients/image18.png "Получатели") <center><u>Получатели - фото 18</u></center>

Дальнейшие действия подобны добавлению номеров при [добавлении адресной книги](#добавление-адресной-книги).

#### Экспорт списка номеров

Для экспорта списка номеров, нажмите иконку кнопки **Экспорт**.

![image19](/img/ru/client_address_book_recipients/image19.png "Получатели") <center><u>Получатели - фото 19</u></center>

Отчет будет сохранен на вашем устройстве в формате **.xlsx**.

#### Поиск

Чтобы найти номер, введите его в поле поиска и нажмите иконку кнопки **Поиск** или клавишу **[ Enter ]**.

![image20](/img/ru/client_address_book_recipients/image20.png "Получатели") <center><u>Получатели - фото 20</u></center>

#### Удаление номеров получателей

Чтобы удалить номер получателя, отметьте чекбокс у одного или нескольких номеров и нажмите иконку кнопки **Удалить**.

![image21](/img/ru/client_address_book_recipients/image21.png "Получатели") <center><u>Получатели - фото 21</u></center>
