Данная вкладка содержит историю входа ваших клиентов в систему. Позволяет просмотреть дату входа, IP-адрес устройства с которого выполнен вход, а также детальную информацию про браузер клиента.

Щоб открыть дану вкладку, выберите **Клиенты** → **Клиенты** в боковом меню и перейдите на вкладку **ПОСЕЩЕНИЯ**.

![image1](/img/ru/admin_clients_visits/image1.png "Посещения") <center><u>Посещения - фото 1</u></center>

## История входа

Таблица содержит историю входа ваших клиентов в систему. Данные расположены в обособленных колонках.:

![image2](/img/ru/admin_clients_visits/image2.png "Посещения") <center><u>Посещения - фото 2</u></center>

|  №  | Колонка | Описание |
| :-: | ------- | -------- |
| 1 | **Дата** | Дата и время входа клиента в систему |
| 2 | **IP** | IP-адрес устройства с которого выполнен вход |
| 3 | **Браузер** | Подробная информация о браузере клиента |
